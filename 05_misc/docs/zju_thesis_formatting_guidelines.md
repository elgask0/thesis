### 1.1.1 Heading 3

> **Level-3 heading**: Arabic numerals in Times New Roman; Chinese text in FangSong; bold; Small Fourth size; 1.5-line spacing; 0.5 line before and 0 line after.

Figures, tables, formulas, and similar items should all be numbered consecutively by chapter using Arabic numerals, for example: Figure 1-3 [the 3rd figure in Chapter 1], Table 2-1 [the 1st table in Chapter 2], and (3-2) [the 2nd formula in Chapter 3]. A spacing of 0.5 line should be left between figures, tables, formulas, and the main text (adjusted through paragraph spacing before/after).

Figures must have figure titles, and tables must have table titles, placed after the figure/table number respectively. The figure number and title should be centered below the figure, in FangSong, 5-point font, single spacing, with 0 lines before and 0.5 line after. The table number and title should be centered above the table, in FangSong, 5-point font, single spacing, with 0.5 line before and 0 lines after. If a figure or table is cited from another source, the literature source should be indicated in the upper-right corner of the figure title or table title.

If a figure or table contains notes, they should be numbered in lowercase English letters in sequence, and the notes should be placed below the figure or table.

## Figures

1. Figures must appear immediately after the relevant discussion. In the main text, the figure number and content should generally be mentioned before the figure itself appears. As a rule, figures should not appear before they are mentioned; where exceptional delay is necessary, the figure should not cross into another section.
2. Photographs provided should be of appropriate size, with a clear subject, distinct layers, and must include a scale bar.
3. Figures should be **self-explanatory**. That is, the meaning of the figure should be understandable from the figure, title, and legend alone, without reading the main text.

A commonly used simplified type of functional graph is called an abbreviated function graph, for example: **Figure 1-1**. (In Word, links can be set through: References → Caption → Cross-reference.)

Axis labels in figures explain the physical meaning of the coordinates. They consist of the symbol or name of the physical quantity together with its corresponding unit. Symbols for physical quantities should be in italics; unit symbols should be in roman type. Quantity and unit should be separated by a slash, for example: *I/A*, *ρ/kg·m^-3*, *F/N*, *υ/m·s^-1*, increase/%.

4. Text inside figures should be in 5-point font. If the arrangement is too dense, a smaller size may be used, but it must not be smaller than Small 7-point.

**Figure 1-1** Increase in the total number of SCI-e publications in 2005 relative to 2001 for five universities

5. Figures should generally be sized **6.67 cm wide × 5.00 cm high**. In special cases, **9.00 cm × 6.75 cm** or **13.5 cm × 9.00 cm** may also be used. In any case, figures of the same type within one thesis should be consistent in size, aesthetically arranged, and neatly aligned. Figures should have **no border**.
6. If a figure contains several subfigures, each subfigure should be labeled sequentially as (a), (b), (c), ... . Each subfigure caption should appear directly below its respective subfigure, and the overall caption should be centered below all subfigures together, as shown below.

## Tables

Tables should be designed to follow the relevant discussion closely. Table layout should generally be read horizontally from left to right for content and test items, and vertically for data in sequence. Tables should be self-explanatory. Large tables or tables used as tools may be placed in the appendix as attached tables. Parameters in thesis tables should indicate the symbols for quantities and units.

1. The table header and the main body of the table must not be split across pages. The header must not appear at the bottom of one page while the table body begins on the next. If only one or two lines remain on the previous page, place the entire table on the next page and leave blank space at the end of the previous page.
2. All physical quantities and dimensions in tables must be labeled according to international standards (SI) and the legally prescribed national symbols and units.
3. **Three-line tables** must be used throughout, aligned to the text width. The top and bottom lines should be thicker, while the middle line should be 1 pt or 0.75 pt (the top/bottom lines and middle line should be visually distinguishable depending on print effect). See Table 1-1.
4. When using another person's table, the source must be indicated.
5. Text in tables should be in 5-point font. If the layout is too dense, a smaller size may be used, but not smaller than Small 7-point.
6. Table headers should be in 5-point font, single spacing, with 0.5 line before and 0 lines after.
7. Tables must span the full text width; that is, the table width must align with the text block, with no vertical lines, as shown in **Table 1-1**.

## Formulas

1. Formulas should start on a new line and be centered. Long formulas should, where possible, be broken after the equals sign, or after symbols such as “+” and “−”. In fractional expressions, the lengths of horizontal fraction bars should be clearly distinguished; the main horizontal bar should align with the equals sign.
2. Formulas should be numbered. The formula number should be enclosed in parentheses and use the format **chapter number–formula number**. For example, formula **(2-3)** is the 3rd formula in Chapter 2. The number should be placed at the far right end of the line, with no dotted leader, in Times New Roman, Small Fourth size.
3. Explanations below formulas should begin with **“Where:”** flush left, followed by the symbols and their explanations. Explanations should be ordered from left to right and top to bottom, separated by semicolons.
4. All physical quantities and dimensions in formulas must be labeled according to international standards (SI) and the legally prescribed national symbols and units. Obsolete symbols and units are prohibited.

## References

The heading **“References”** should be centered, in bold FangSong, Third size, with 1 line before and 0.5 line after. All cited references should be listed uniformly after the main text.

1. If citing a viewpoint from a journal article, the format is: “a viewpoint or conclusion (Author, YYYY)” or “Author (YYYY) argues that ...”, where **YYYY** is the four-digit publication year. If the same author published more than one article in that year, they should be distinguished by **a, b, c**, etc., in the order of publication, for example **YYYYa** or **YYYYb**.  
   If citing a viewpoint from a book, the format is: “a viewpoint or conclusion (Author, YYYYa, P.xxx)” or “Author (YYYYb, P.xx–xx) argues that ...”. Here, **P.xxx** indicates the page number where the cited view appears, for example, **P.432**.
2. **Reference-listing format at the end of the thesis** (for Chinese references: FangSong, Small Fourth size, 1.5-line spacing, 0 lines before and after, hanging indent, justified alignment):

   - **Journal article**: [No.] Author. Title [J]. Journal Name, Year, Volume(Issue): Start page–End page.  
   - **Book**: [No.] Author. Book Title [M]. Place of Publication: Publisher, Year: Start page–End page.  
   - **Collected papers / proceedings**: [No.] Author. Title [A] (or Editor, Collection Title [C]). Place of Publication: Publisher, Year: Start page–End page.  
   - **Dissertation / thesis**: [No.] Author. Title [D]. Doctoral Dissertation (or Master’s Thesis), Degree-granting Institution, Year.  
   - **Electronic document**: [No.] Author. Title of Electronic Document [Electronic document and carrier type code]. Source or accessible address, publication or update date / citation date (optional).

   Codes for electronic documents and carriers: **DB/OL** = online database, **DB/MT** = magnetic tape database, **M/CD** = CD-ROM book, **CP/DK** = disk software, **J/OL** = online journal, **EB/OL** = online electronic bulletin. (English electronic references should appear after all other English references; Chinese electronic references should appear after all other Chinese references.)
3. **English references** should follow the same rules, in Times New Roman, Small Fourth size, 1.5-line spacing, 0 lines before and after, hanging indent, justified alignment. Author names must follow the order **surname first, given name after; surname not abbreviated, given name abbreviated**. For example, the format for a journal article is: “Surname, Initial(s). Article title [J]. Journal name, Year, Volume(Issue): Start page–End page.”
4. In the reference list, punctuation should all be in English style. However, English references use half-width punctuation, while Chinese references use full-width punctuation (except that the “document type identifiers” in Chinese references should use half-width English punctuation, such as **[J] [D]**, not full-width **［Ｊ］［Ｄ］**).
5. The entire reference list should be arranged with **English references first** and **Chinese references second**. English references should be ordered alphabetically; Chinese references should be ordered according to the alphabetical order of **Hanyu Pinyin**.
6. For works with **three or fewer authors**, all authors should be listed. In Chinese this is “***, *** and ***”; in English this is “***, *** & ***”. For works with **more than three authors**, list only the first three; in Chinese add **“et al.” equivalents (“等”)**, and in English add *et al.* in italics. Whether in foreign languages or Hanyu Pinyin, author names must uniformly appear as **surname first, given name after**.
7. Students are asked to carefully check whether every reference contains complete bibliographic elements, whether punctuation is correct, and whether fonts and formatting comply with the requirements. Please do **not** export references directly from a website and paste them without checking.
